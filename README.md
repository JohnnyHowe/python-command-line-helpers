# python-command-line-helpers

Helper utilities for Python command-line tooling.

## Install From GitHub

```bash
pip install "python-command-line-helpers @ git+https://github.com/OWNER/REPO.git"
```

## Session-Only Install (venv)

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install "python-command-line-helpers @ git+https://github.com/OWNER/REPO.git"

python3 path/to/your/script.py

deactivate
```

## Development

Run tests from repo root:

```bash
python3 tests/run_all.py
```
